## ‚ùì **TROUBLESHOOTING**

### **"Failed to extract text" (Non-OCR)**
- **Cause:** PDF might be scanned or image-based
- **Solution:** Switch to OCR mode

### **"OCR is taking too long"**
- **Cause:** Large PDF with many pages
- **Solution:** This is normal. Wait for completion or split the PDF first

### **"Garbled or missing text"**
- **Non-OCR mode:** PDF might be protected or encrypted
- **Solution:** Try OCR mode or use password removal tool first

### **"OCR produced incorrect text"**
- **Cause:** Low-quality scan or unclear text
- **Solution:** Re-scan document with better quality, or manually correct in Word

### **Browser crashes with large files**
- **Cause:** Large PDF with OCR uses a lot of memory
- **Solution:**
  - Close other tabs
  - Split the PDF into smaller chunks first
  - Use Non-OCR mode if possible
