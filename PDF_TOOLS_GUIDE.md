## â“ **TROUBLESHOOTING**

### **"Failed to process PDF"**
- Ensure the file is a valid PDF
- Try opening it in a PDF reader first
- Check if it's password-protected

### **"Page does not exist"**
- Double-check your page range
- Verify the PDF has that many pages
- Use "1-3" format, not "1,2,3" for ranges

### **Browser crashes with large files**
- Close other browser tabs
- Try a smaller PDF first
- Split large operations into smaller batches

### **Compressed file is same size**
- PDF may already be optimized
- Image-heavy PDFs compress less
- This is normal for some PDFs
