## â“ **TROUBLESHOOTING**

### **"Please select a Word document (.docx file)"**
- **Cause**: You uploaded a .doc or other file format
- **Solution**:
  1. Open the file in Microsoft Word
  2. Go to File â†’ Save As
  3. Choose "Word Document (.docx)" format
  4. Save and try again

### **"Failed to parse document"**
- **Cause**: Corrupted or password-protected file
- **Solution**:
  1. Try opening in Word to verify it's not corrupted
  2. Remove password protection if present
  3. Save a new copy and try again

### **"Missing text in PDF"**
- **Cause**: Complex formatting or embedded objects
- **Solution**: This is rare, but if it happens:
  1. Simplify the Word document
  2. Remove complex elements (SmartArt, charts)
  3. Convert again

### **"PDF looks different from Word"**
- **Expected**: Formatting is simplified
- **Explanation**: This tool focuses on text content
- **Alternative**: Use "Print to PDF" in Word for exact formatting
